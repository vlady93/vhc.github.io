SISTEMA DE LIQUIDACIONES


Empresa: Arcis Veritas Trading Agent S.R.L.

El sistema fue diseñado para automatizar el proceso de liquidaciónes que realiza la empresa. Incluye módulos para administrar clientes, términos, penalidades, liquidaciones y usuarios con roles específicos. Su objetivo principal es generar y ver las liquidaciones en PDFs, lo que permite tener un control preciso y eficiente de todas las liquidaciones realizadas en la empresa. El sistema se desarrolló con tecnologías PHP framework Laravel, además de JavaScript, Bootstrap y jQuery.
