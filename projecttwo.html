SISTEMA DE CONTROL DE PROYECTOS DE GAS


Empresa: GEDEKAGAS

El sistema fue diseñado para tener un control del inventario y proyectos de la empresa. Incluye módulos para administrar proyectos, proveedores, categorías, ciudades, materiales, ingreso y salida de material, clientes, reportes y usuarios con roles específicos. Su objetivo principal es llevar un seguimiento preciso de los proyectos, el personal asignado y el material utilizado. El sistema se desarrolló con tecnologías PHP y el framework Laravel, además de JavaScript, Bootstrap y jQuery.
